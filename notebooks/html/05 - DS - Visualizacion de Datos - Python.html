Definición¶

Inicio

Data Visualization es la prctica de convertir datos crudos a representaciones grficas como mapas, grficas, grafos y dashboards. Hace a un dataset complejo, ms accesible, entendible y utilizable.

¿Porqúe hacemos tablas y gráficos?¶

Inicio

Para entender cun significativa puede ser una correcta visualizacin de datos, debemos remitirnos a la propia naturaleza humana y a la ciencia que estudia en particular como funciona el cerebro humano: la Neurociencia.

La Neurociencia, ha demostrado que el cerebro humano le cuesta mucho ms entender los datos complejos cuando se codifican en nmeros y texto en comparacin a utilizando representaciones visuales y grficas. Las tcnicas de Dataviz aprovechan esa calidad, ayudando a presentar cantidades masivas de informacin en la forma ms eficiente y visual.

En particualr, es muy til en la etapa de comprensin de los datos, armar distintas visualizaciones utilizando tablas y grficos de distinto tipo (puntos, barras, torta), seleccionando atributos y graficandolos para analizar su distribusin y relacin con otros atributos del dataset.

Utilizando el tipo correcto de visualziación¶

Inicio

A lo largo del tiempo, han surgido variedades de tcnicas de visualizacin de datos, herramientas y tpos de grficos, abriendo oportunidades para resolver diferentes tarea de un cientfico de datos. Por esta razn, dependiendo de los datos que se quieren explorar, la naturaleza de ellos y las hiptesis que deseamos validar, la tcnica o herramienta que utilizaremos.

En particular, es muy importante elegir el tipo de visualziacn correcta, para alcanzar el xito de un proyecto de de data science. Por ejemplo, si el to de grfico elegido no es el correcto, podemos omitir un problema de sesgo en un conjunto de datos o podramos no trasmitir correctamente las conclusiones de un anlisis de datos al cliente final.

Tareas en Data Visualziation:

Una forma muy fcil de separar las distintas tcnicas de visualizacin de datos disponibles, es de acuerdo a la tarea que tenemos como objetivo.

1. Comparar Datos¶

Para comparar datos, por ejemplo de diferentes variables de un mismo dataset, o datos de un dataset contra otro, los siguientes tipos de grficos son muy tiles:

  • Barras: Permite comparar de forma directa, valores cuantitativos por variable o categora
  • Pilas apliadas: Permiten analizar valores cuantitativos por categora y a la vez compararlos con diferentes muestras. Por ejemplo analizar el PBI de diferentes paises y teniendo en cuenta el aporte porcentual de cada sector.
  • Grfico Radar: Es otra alternativa para comparar los valores cuantitativos de diferentes variables. Por ejemplo, mostrar el puntaje en diferentes habilidades que hace a la destreza de un jugador de futobl, como Conversin de Goles, Precisin en pases, etc.

2. Explorar relaciones Part-To-Whole (de la parte al todo)¶

Cuando lo que se quiere es explorar la composicin por ejemplo de una variable categrica y analizar relaciones del tipo part-to-whole, los siguientes tipos de visualziaciones son muy tiles:

  • Grfico de Torta (Pie chart/Donut chart):Permite analizar de forma bsica de la composicin porcentual de una variable categrica.
  • Grfico Piramidal (Pyramid chart): Cuando los datos representan o responden a una "jerarqua", puede ser interesante utilizar este tipo de grficos.
  • Mapa de rbol (Treemap chart): Permite representar relaciones jerrquicas complejas. Por ejemplo la cantidad de ingresos anuales de una compaa agregados por unidades de negocio y para cada unidad de negocio por proyecto.
  • Grfico de Embudo (Funnel chart): Este tipo de grfico se utiliza mucho para comparar variables cuantitativas, que de alguna forma estan relacionadas entre si como parte de un proceso. Por ejemplo, se puede utilizar para analizar ventas de un sitio de eComerce, ayudando a identificar los cueyos de botella del proceso.

3. Analisis en el tiempo¶

Cuando se busca analizar la relacin entre una o ms variables con una componente temporal, como por ejemplo puede ser analizar la evolucin de los salarios de Uruguay, respecto del tiempo, los siguientes tipos de visualizaciones son los indicados:

  • Grfico de linea/curva (Line/Spline chart): Es la representacin grfica ms simple y utilizada para mostrar la evolucin de una variable cuantitativa en funcin de otra variable.
  • Grfico de rea (rea chart): A diferencia de la anterios, utiliza reas para representar una variable cuantitativa.
  • Grfico candela Japonesa (Candlestick chart): Es muy til cuando se queire analizar una variable que puede tomar mltiples valores para un mismo momento, por ejemplo analizar la evolucin del precio de las acciones de una compaia frente a otras, tomando como referencia el precio de apertura y cierre de las acciones en la bolsa por da. Es una variante de las grficas OHLC.
  • Grfico Sparkline (Sparkline chart): Consiste en construr versiones simplificadas y pequeas de otros grficos temporales como Line Charts o rea Charts, eliminando por ejemplo los ejes del grfico. Son muy tiles para utilizar en tablas, en donde para cada fila queremos hacer ms visual una variable determinada. Por ejemplo listar los montos totales de exportaciones de granos en Uruguay en un ao entero, utilizando Sparklines para mostrar para cada grano la variacin mensual del precio de este.

4. Analizar distribución de un dataset¶

Para analizar la distribucin de una variable en un determinado dataset y comparar por ejemplo con otros datasets, encontrar relaciones y co-relaciones, los siguientes tipo de grficos son muy tles:

  • Histogramas: Un histograma es una representacin de la estimacin de la distribucin de valores de un dataset numrico.
  • Grfico de dispersin de puntos (Dot (scatter) chart): Permite examinar las tendencias en distribucin y correlacin entre dos variables.
  • Grfico de burbujas (Bubble charts): Cada punto se grafica como un disco, expresando dos de los atributos/dimensiones del dato a travs de la ubicacin del disco en el sistema de ejes cartesianos y el tercero a travs de su tamao.
  • Grfico Box-and-whisker
  • Grfico de errores
  • Mapa de calor (Heatmap chart): Es una representacin visual bidimensional de los datos, donde los valores se codifican en colores.
  • Grfico de rangos (Range chart)
  • Grfico Polar (Polar chart)

5. Evaluar performance de un dato¶

Cuando se quiere evaluar el rendimiento de una variable y comparar el valor real con valores KPI se suele recurrir a los siguientes tpos de grficos:

  • Gauge Ciruclar
  • Gauge Lineal
  • Bullet Graph

6. Datos geográficos¶

Muchas veces, la mejor forma de darle sentido a datos geogrficos (geo-referenciados) es utilizando visualziaciones en un mapa.

  • Mapa de Colores(Choropleth maps):Permiten identificar diferencias entre reas geogrficas.
  • Mapa de Puntos(Dot maps):Proporcionan una visualizacin accesible de la distribucin geogrfica de los fenmenos en las reas geogaficas en la que se encuentran.

  • Mapa de Burbujas(Bubble maps):Similar a los grficos de burbujas, permiten agregar una tercera dimensin a la posicin de un fenmeno, utilizando el tamao de una burbujar. Son muy tiles para agregar la informacin de un mapa de puntos cuando contamos con tantos puntos que se hace dificil entender la visualizacin.

  • Mapa de Conexiones(Connector map): Permite representar relaciones entre los distintos puntos del mapa mediante lineas que unen los puntos.

  • Mapa de Flujos(Flow map): tiles para explorar cmo los objetos se mueven entre ubicaciones, en especial cuando interesa mostrar la direccin del flujo.

7. Análisis de Textos¶

De la mano de la explosin de las redes sociales, ha explotado todo lo relacionado al anlisis de datos en este tipo de redes, apareciendo nuevas reas enfocadas especficamente en el anlisis de este tipo de datos. Por ejemplo Social Listening o Social Media Measurement y Social Business Inteligence (SBI), entre otros.

Uno de los requerimeintos ms simples en este tipo de analisis es cuantificar de que se est hablando en las redes y lograr comunicar esto con visualziaciones intuitivas, por ejemplo utilizando nubes de palabras.

  • Nube de Palabras (Word-Cloud): Muestra los diferentes valroes de una variable categrica, en particular una variable de texto libre, donde para cada palabra se agrega una tercera dimensin que es su tamao. Usualmente se cuenta la cantidad de apariciones de la palabra dentro del dataset,y ese valor es utilziado como peso de la categora, para determinar el tamao.

Film Dialogue¶

Film dialogue es un proyecto de HANAH ANDERSON y MATT DANIELS, quienes buscaron estudiar que tan homogenea es la participacin de actores femeninos y masculinos en los guiones de pelculas de Hollywood.

En el sitio oficial, puede verse a traves de una serie de visualziaciones interactivas y muy intuitivas, la participacin en cantidad de dilogos entre actores hombres y mujeres para ms de 2.000 guines de peliculas famosas.

Extrado del sitio oficial del proyecto:

Lately, Hollywood has been taking so much shit for rampant sexism and racism. The prevailing theme: white men dominate movie roles. But its all rhetoric and no data, which gets us nowhere in terms of having an informed discussion. How many movies are actually about men? What changes by genre, era, or box-office revenue? What circumstances generate more diversity?

Veamos algunas de estas visualizaciones para comprender un poco mejor de lo que estamos hablando. En la siguiente grfica interactiva, se muestran una cantidad de datos bien interesantes. Lo que esta graficado es la distribucin de palabras para cada pelicula, entre hombres y mujeres. En el extremo izquierdo tenemos aquellas peliculas en las que los dialogos son exclusivamente de actores masculinos, mientras que en el extremo derecho tenemos las peliculas cuyos dilogos pertenecen exclusivamente a actrices femeninas. Luego a medida que nos alejamos de los extremos podemos ver las peliculas cuya distribucin de dialogos entre ambos gneros varian.

Por otro lado, el largo de la barra en la que se representa la distribucin no tiene una escala constnte, si no que contempla la cantidad de peliculas asociadas a cada distribucin. De esta forma si hay ms peliculas en las que dominan los dilogos en un gnero sobre otro, el centro del gafico (punto donde la distribucin es 50/50) no estar en el centro del gfico, si no que por el contrario estar ms cerca de uno de los extremos.

Para empezar, notese cuan claro es a la hora de representar que tan equitativo o no, son los guinoes de las pelculas. Segundo, cuan claro es para condensar mucha informacin en una sola imagen, por ejemplo la distribucin de los dialogos por gnero minuto a minuto, para una misma pelcula.

Referencia: https://pudding.cool/2017/03/film-dialogue/

After Babylon¶

After Babylon es un proyecto OpenSource que surge como trabajo laboratorio del curso Density Design, el cual forma parte de un programa de formacin y especializacin de post grado en Diseo de Comunicacin, en el Politcnico di Milano. Este trabajo toma datos de la base de datos abierta WALS.

WALS por la sigla "The World Atlas of Language Structures" es una base de datos estructurada con propiedades de una enorme variedad de lenguajes a lo largo y ancho del mundo (propiedades phonticas, gramaticales y sintcticas), generado por un equipo de ms de 55 especialistas en la materia.

Dentro del sitio del proyecto se pueden ver varias visualziaciones bien interesantes, como el lugar origen de cada uno de los 2.678 lenguajes vivientes que tienen en el dataset analizado (ver figura debajo).

Por otro lado, notar como este equipo resolvi representar la interaccin entre dos lenguas de diferentes paises, cuando se da el fenmeno de "tomar prestado" palabras de otro lenguaje. Por ejemplo cuando incorporamos al espaol la palabra "sandwich" para referenrnos a un snguche.

Referencia: http://www.puffpuffproject.com/languages.html

SELFIECITY¶

Selfiecity es una iniciativa que investigo caractersticas de "selfies" a lo largo de 5 pases diferentes, utilizando una mezcla de tcnicas tanto matemticas como artsticas. Realmente, no solo vale la pena ver los resultados y variar de locacin para ir conociendo ms acerca de los resultados obtenidos para cada una, si no apreciar el ingenio con el que se resolvieron algunas visualizaciones y sobre todo la interaccin con la persona quien las consulta.

Por ejemplo, en la siguiente visualizacin, se puede ver la distribucin de "selfies" organizada por paises y graficada segn la edad inferida para cada selfie. A su vez, los resultados se presentan organizados por gnero. De esta forma, por ejemplo podemos comparar cual es la edad media de las personas que se toman selfies tanto por gnero como por ubicacin.

Otro de los datos que se analizan y muestran de forma similar a la visualizacin anterior, es la distribucin de las selfies de acuerdo al sentimiento de la foto: positivo, neutral y negativo. Por ejemplo, de esta representacin grfica, rapidamente podemos inferir que en el dataset analizado, las selfies ms "felices" se toman en la ciudad de Sao Paulo y en particular son tomadas por mujeres. Mientras tanto, mosc es la ciudad ms seria en lo que a selfies concierne, tanto hombres como mujeres.

Referencia: http://selfiecity.net
